HumanMap
  • Auto-adjusting results
  • Drawing shapes
This app is about geolocalized data and how to handle it with MongoDb Geospatial Indexing
Each marker is a place in Paris where a film was shot in the last 4 years.

I built it using MongoDb, Jongo, SpringMVC, Angularjs, OpenStreetMap and Leaflet.
Feel free to fork it on Github !
